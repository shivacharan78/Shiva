ROLLUP,CUBE:-Oracle 8i introdused rollup,cube clauses.these clauses
============  are optoinal clauses used along with group by clause
only.these clauses are used to calculate subtotal and grand total.


Syntax:- select col1, col2..... 
=======   from tablename
         group by rollup(col1,col2....);


Syntax:- select col1, col2....
========  From tablename 
          group by cube(col1,col2....);


Rollup is used to calculate subtotal and grandtotal based on a single column.
where as if we want to calculate subtotal,grandtotal based on all grouping
columns then we are using cube.

Eg:- 
====
