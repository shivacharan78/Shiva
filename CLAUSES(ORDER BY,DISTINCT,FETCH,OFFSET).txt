                                 ORDER BY CLAUSE
                                ==================

Order by:-order by clause is used to sorting data either ascending 
========  order or descending order.
     
order by clause having two keywords.
1)asc
2)desc

syntax:-
=======
    select * from tablename
    order by columnname[asc/desc];
