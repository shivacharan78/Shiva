                                SEQUENCE
                              ===========

     Sequence is an database object which is used to generates 
Sequence of numbers automatically. 

      Generally sequences are used to generates Primary key values automatically. 
Generally sequence is an independent database object.That's why Once a sequence 
has been created then number of users simultaneously access that sequence.

   in sql server if we want to generate first sequence number then
we must use next value pseudo column.

syntax:-
=======
select next value for sequencename;

***Generating primary key values automatically(Auto-increment):-

     In all relational databases generating primary key values automatically 
is called auto-increment concept.in oracle we are implementing auto-increment
concept by using sequence database object.
