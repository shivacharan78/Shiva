Task 1: Library and Book Injection
Objective:
Create a Spring Core application where a Library class contains a single Book object injected via setter injection.

Requirements:

Book class:

String title

String author

Setter methods.

Library class:

Book book

Setter method for book.

showBookDetails() method to display book info.

Configuration:

Use @Configuration and @Bean to define both beans.

Main Class:

Load context, get Library bean, and display book details.
