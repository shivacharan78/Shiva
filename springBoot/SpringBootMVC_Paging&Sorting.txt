Student Records Management System (CRUD + Paging & Sorting)
🎯 Objective
Build a Spring Boot MVC application to manage student records with CRUD operations, pagination, and sorting.

✅ Requirements

Technologies

Spring Boot 3.1.5

Spring Web (MVC)

Spring Data JPA

Thymeleaf or JSP

MySQL / PostgreSQL

Lombok (optional)

Functionalities

Add Student

Fields: Roll No, Name, Department, Semester, GPA, Email.

View Students List (Paginated, default 10 per page).

Sortable columns: Name, GPA, Department, Semester.

Edit Student

Delete Student

Search by Name or Department.

Endpoints
| Method | Endpoint | Description |
|--------|------------------------|-----------------------------------|
| GET | /students | List students with paging & sorting |
| GET | /students/new | Show add form |
| POST | /students | Save student |
| GET | /students/edit/{id} | Show edit form |
| POST | /students/{id} | Update student |
| GET | /students/delete/{id} | Delete student |
