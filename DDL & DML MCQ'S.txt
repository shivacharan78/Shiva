1.DDL statements are used for which of the following database objects ?

a) Tables

b) views    

c) indexes   

d) All of the above
	

Answer: DDL works on tables, views, indexes, schemas, etc.

2.What does the TRUNCATE command do in SQLSERVER?

a) Deletes table and its structure

b) Removes all rows but keeps the structure

c) Deletes selected rows with WHERE clause

d) Drops only column data


Answer: b) Removes all rows but keeps the structure
👉 TRUNCATE deletes all records, structure remains.

📌 Quick Memory Tip

DDL → CREATE, ALTER, DROP, TRUNCATE

DML → INSERT, UPDATE, DELETE, MERGE

SELECT → DQL

COMMIT / ROLLBACK → TCL
